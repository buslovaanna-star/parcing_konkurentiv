# ProcureAI — Аналіз закупівель

## Структура репозиторію
```
app.py                  ← головний додаток
requirements.txt
data/
  barcodes.csv          ← статичний маппінг: штрихкод → артикул iHerb (не змінюється)
```

## Файли що завантажуються щоразу (5 шт.)
| Файл | Що містить |
|------|-----------|
| Прайс iHerb | артикул iHerb, ціна USD, наявність |
| Прайс VitaWorld | артикул VW, ціна USD, штрихкод |
| Прайс DSN | артикул DSN, ціна UAH, штрихкод в описі |
| Продажі + залишки | артикул iHerb, дата, кількість, залишок |
| РРЦ | артикул iHerb, дата, ціна UAH |

## Розгортання на Streamlit Cloud
1. Завантажте репозиторій на GitHub
2. Перейдіть на https://streamlit.app → New app
3. Вкажіть репозиторій і файл `app.py`
